# Project Overview
Finance Manager Pro is a cross-platform personal finance management application built in Java.
The application allows users to:
- Track income and expenses.
- Categorize transactions.
- Create monthly budgets.
- Monitor spending habits.
- Calculate loans and mortgages.
- Receive AI-powered financial insights.

Targeted Platforms-
- macOS
- Windows

Technology Stack-
- Java 25
- SQLite
- JDBC
- JavaFX
