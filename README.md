# Finance-Manager-Pro
Project Status: (docs/project-status.md)

A personal finance management application that helps users track expenses, manage budgets, calculate loans/mortgages, and receive AI-powered financial insights.

**Purpose-** This application has been designed to help users effectively manage, monitor and improve their financial well-being through a combination of expense tracking,
budget tools, financial calculator and intelligent insights. The primary purpose of the application is to provide individuals with a centralized platform where they can record their income and expenses, categorize transactions, monitor spending habits, and gain a clear understanding of their overall financial position. Many people struggle with managing personal finances due to a lack of organization, limited financial knowledge, or difficulty visualizing where their money is being spent. Finance Manager Pro aims to address these challenges by offering a simple, intuitive,
and user-friendly solution that enables users to make informed financial decisions based on accurate and up-to-date information.

**Type-** Desktop Application

**Operating System-** macOS and Windows

**Language-** Java

**UI Framework-** JavaFX

**Database-** SQLite

**Targeted Users-**
1. Students
2. Young Professionals
3. Families
4. Anyone wanting to manage personal finances.

**Platform Compatibility**
This application has been completely developed on macOS using Java and JavaFX, however, the app intends to be fully compatible woith both macOS and Windows. This app has been tested on macOS during development and packaged for both macOS and Windows.
